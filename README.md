# Student Record System

Student Record System is a Python-based project designed to manage student information efficiently. It allows users to store, view, update, and manage student records through a simple interface.

## Features

- Add student records
- View student details
- Update student information
- Manage marks and grades
- Display stored records

## Technologies Used

- Python

## Learning Outcomes

- File handling
- Functions
- Conditional statements
- Loops
- Data management
